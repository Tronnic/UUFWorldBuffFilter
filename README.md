# UUF World Buff Filter

<p align="center"><img width="300" height="300" alt="image" src="https://github.com/user-attachments/assets/a0f62f98-fa39-488c-831e-960106d69653" /></p>



UUF World Buff Filter allows you to hide unwanted world and miscellaneous buffs from the Player frame in Unhalted Unit Frames (UUF) using a customizable blacklist.

It is designed for players who want a clean buff display and prefer to see only relevant combat, class, or raid buffs while hiding temporary world effects, holiday buffs, zone effects, or other clutter.

Note: This does not work for buffs you get while in combat. Buffs you had before combat will stay hidden  (Timewalking, Long-Duration buffs, Tabard etc.)

---

## What It Does

This addon hooks into UUF’s Player Buff container and filters out specific buffs.

If a buff is on your blacklist, it will not be shown in UUF’s Player frame.

Also hides Blizzard's default buff & debuff frame. (It's still there when you mouse-over it)

---

## Features

- Buff-blacklist with individual remove buttons  
- Dropdown menu to add currently active buffs  
- "Blacklist all current buffs” button  
- “Clear blacklist” button  
- Hide Blizzard Default Buffs & Debuffs  
- Automatic Settings lock while in Combat or Instance  
- Minimap-Button to open WBF Settings and UUF Settings  

---

## Settings Location

You can access the graphical settings here:

- Minimap Button  
- Esc → Options → AddOns → UUF World Buff Filter

## Important Notes

This addon only affects the PlayerFrame Buffs in UUF. 

You should NOT change settings during combat or in dungeon/raid due to Blizzards new restrictions.

You can get Unhalted Unit Frames here: https://www.curseforge.com/wow/addons/unhaltedunitframes
